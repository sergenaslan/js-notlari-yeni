.map() Fonksiyonu

ES5
Verilen dizi elemanlarına, girilen callback fonksiyonu uygular ve elde edilen sonuçlarla yeni bir dizi oluşturur. 

const urunler = [
    { id: 1, name: "Kalem", fiyat: 5 },

    { id: 4, name: "Kalemtraş", fiyat: 7 },
];

Arrow Function (=>)

ES6

function add(a, b) {
  return a + b;
}

// Arrow function

console.log(add2(2, 3));  // 5

//Not eğer arrow funciton içersindeki işlem tek bir 
//satırlık işlem ise aşşağıdaki gibi kısaltılarak kullanılabilinir.

structuredClone()

ES11
JavaScript’te structuredClone(), bir objenin tam kopyasını (deep copy) oluşturmak için kullanılan yerleşik bir fonksiyondur.
  • Orijinal nesnenin tüm iç yapısını yeni bir nesneye kopyalar
  • İç içe (nested) objeler ve array’ler aynı referansı paylaşmaz
  • Tamamen bağımsız bir kopya oluşur
Bu yüzden deep copy yapmak için kullanılan modern ve güvenli yöntemlerden biridir. 

const user = {
  name: "Sergen",
  address: {
    city: "Istanbul"
  }
}

const copy = structuredClone(user);

copy.address.city = "Ankara";

console.log(user.address.city); //Istanbul
