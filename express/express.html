Express Kütüphanesi

1. Express neden var?

Önce bunu anlayacağız. Node tek başına HTTP server açabiliyor:


const http = require('http');

const server = http.createServer((req, res) => {
  res.end('Merhaba');
});

server.listen(3000);

Bu çalışır. Ama problem:

  • routing yok
  • middleware yok
  • body parsing zor
  • query parsing uğraştırır
  • error handling dağınık
  • kod büyüdükçe kaos

Express bunun üzerine abstraction koyar.Yani: Node HTTP modülünün daha kullanışlı hali.

const app = express();
