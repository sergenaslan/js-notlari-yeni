js class özelliği

Instance = new anahtar kelimesiyle bir sınıftan (veya constructor fonksiyondan) türetilmiş nesneye denir

1. Sınıf Tanımı (Class Declaration / Expression)


// Declaration
// ✅ Declaration kullan:
// - Basit, tek bir sınıf tanımı
// - Kodun başında net bir şekilde görünmesini istiyorsan
class Araba {
  constructor(marka, model) {
    this.marka = marka;
    this.model = model;
  }
}

// Expression
// ✅ Expression kullan:
// - Koşullu/opsiyonel sınıflar
// - Factory function'lar
// - Tek kullanımlık/anonim sınıflar
// - Higher-order function'larda
const Hayvan = class {
  constructor(tur) {
    this.tur = tur;
  }
};

2. constructor Metodu

Sınıftan new ile nesne oluşturulduğunda ilk çalışan metottur. Nesne özelliklerini başlatmak için kullanılır. 


class Kisi {
  constructor(isim, yas) {
    this.isim = isim;   // özellik (property)
    this.yas = yas;
  }
}

3. Özellikler (Properties)

Instance Properties: this üzerinden tanımlanır, her nesneye özgüdür.
Class Fields (ES2022+): Doğrudan sınıf gövdesinde tanımlanabilir. 

class Ornek {
  alan1 = "varsayılan";         // public instance field
  static alan2 = "statik";      // static field
  #ozelAlan = "private";        // private field (ES2022)
}

1. Instance Field (Örnek Alanı) Nedir?

Instance field, sınıftan new ile oluşturulan her bir nesneye ait olan alandır. Her nesne bu alanın kendi kopyasını taşır. Oluşturulan her bir nesne bu değişken üzerinde farklı veriler tutabilir. 

class Ornek {
  alan1 = "varsayılan";  // instance field
}

const nesne1 = new Ornek();
const nesne2 = new Ornek();

nesne1.alan1 = "değişti";
console.log(nesne1.alan1); // "değişti"
console.log(nesne2.alan1); // "varsayılan" (hala ilk değerinde)
Özellikler:
  1. Her nesne kendine ait bir kopyaya sahiptir.
  2. Bir nesnedeki değişiklik diğer nesneleri etkilemez.
  3. Nesne oluşturulurken her bir instance için ayrı ayrı bellekte yer kaplar.
  4. this üzerinden erişilir.

2. Static Field (Statik Alan) Nedir?

Static field, sınıfın kendisine ait olan alandır. Tüm nesneler tek bir kopyayı paylaşır. Sınıf üzerinden erişilir, nesne üzerinden erişilemez.


class Ornek {
  static alan2 = "statik";  // static field
}

console.log(Ornek.alan2);   // "statik" (sınıf üzerinden erişim)

const nesne = new Ornek();
console.log(nesne.alan2);   // undefined (nesne üzerinden erişilemez!)
Özellikler:
  • Sınıfın kendisine aittir, nesnelere değil.
  • Tüm nesneler paylaşır (tek bir kopya).
  • Bir yerde değiştirilirse, her yerden değişmiş hali görünür.
  • Bellekte sadece tek bir kopya bulunur.
  • Erişim: SınıfAdi.statikAlan şeklindedir.

2.private field

Private field (#ozelAlan), davranış olarak instance field gibi davranır, ancak erişim kontrolü eklenmiş halidir. Yani:
  • Instance özelliği: Her nesnenin kendi kopyası vardır.
  • Static özelliği DEĞİL: Sınıfa ait değildir, nesnelere aittir.
  • Private: Sadece sınıfın içinden erişilebilir, dışarıdan erişilemez.

4. Metotlar (Methods)

Bir sınıfın (class) içinde tanımlanan function anahtar kelimesi kullanılmadan yazılan fonksiyonlara "method" (metot) denir.
  • Bir nesnenin (object) yapabileceği işlemlerdir.
  • Sınıfın davranışlarını (behavior) tanımlar.
  • JavaScript'te sınıf içinde function anahtar kelimesi kullanılmadan yazılır.
  • aşşağıdaki örnek bir

Instance Methodudur

Prototip üzerinde bulunur, tüm nesneler ortak kullanır. INSTANCE METHODLAR sadece instance nesneler üzerinde çalışır class üzerinde çalışmaz 

class Araba {
  constructor(marka, model) {
    this.marka = marka;
    this.model = model;
  }
  
  // İŞTE BUNLAR METOT (method) ↓
  calistir() {
    console.log(`${this.marka} ${this.model} çalıştı.`);
  }
  
  dur() {
    console.log(`${this.marka} ${this.model} durdu.`);
  }
}

const benimArabam = new Araba("BMW", "3.20");
benimArabam.calistir();  // "BMW 3.20 çalıştı." (calistir metodu çağrıldı)
benimArabam.dur();       // "BMW 3.20 durdu."     (dur metodu çağrıldı)

Static Metotlar

Sınıfın kendisine aittir, nesne üzerinden çağrılamaz (örneğin Array.from() gibi).

class Matematik {
  static topla(a, b) {
    return a + b;
  }
}
Matematik.topla(3, 5); // 8

Private Metotlar (#)

Sadece sınıf içinden erişilebilir.

class Gizli {
  #sifreliMetot() {
    return "gizli";
  }
  acikMetot() {
    return this.#sifreliMetot();
  }
} 
let bb = new Gizli();  // Önce nesne oluştur
console.log(bb.acikMetot());  // ✅ "gizllli"

5. Getter ve Setter

1. Getter (get) Nedir?

Bir özelliğin değerini okurken otomatik çalışan metottur. Sanki normal bir özellikmiş gibi nesne.özellik yazarsınız, ama aslında bir fonksiyon çalışır. get ile tanımlanmış fonskiyonları özellikmiş gibi kullanabilmemizi sağlar. 

class Kisi {
  constructor(ad, soyad) {
    this.ad = ad;
    this.soyad = soyad;
  }
  
  // Getter - tamAd okunduğunda bu çalışır
  get tamAd() {
    return `${this.ad} ${this.soyad}`;
  }
}

const k = new Kisi("Efsun", "Can");
//Burada tamAd bir metot gibi görünmez; sanki bir property’ymiş gibi çağrılır.
console.log(k.tamAd); // Efsun Can  (parantez yok! normal özellik gibi)

2. Setter (set) Nedir?

Instance içerisindeki özelliğe değer atarken ekstra validasyon veya veri manipülasyonu yapmak istediğimizde çalışacak, property gibi kullanılan (aslında set ile tanımlanmış) fonksiyonlar yazmak için kullanılır. 

class Daire {
  constructor(yaricap) {
    this._yaricap = yaricap;  // _ ile özel olduğunu belirtiriz (gelenek)
  }
  
  // Getter
  get yaricap() {
    return this._yaricap;
  }
  
  // Setter
  set yaricap(yeniDeger) {
    if (yeniDeger > 0) {
      this._yaricap = yeniDeger;
    } else {
      console.log("Yarıçap pozitif olmalı!");
    }
  }
  
  get alan() {
    return Math.PI * this._yaricap ** 2;
  }
}

const d = new Daire(5);
console.log(d.yaricap); // 5 (getter çalışır)
d.yaricap = -3;         // setter çalışır: "Yarıçap pozitif olmalı!"
d.yaricap = 10;         // setter çalışır, değer değişir
console.log(d.alan);    // 314.159...

6. Kalıtım (extends, super)

Başka bir sınıftan türeyebilir.

class Hayvan {
  constructor(isim) {
    this.isim = isim;
  }
  sesCikar() {
    return `${this.isim} ses çıkarıyor.`;
  }
}

class Kopek extends Hayvan {
  constructor(isim, cins) {
    super(isim); // üst sınıfın constructor'ını çağırır
    this.cins = cins;
  }
  sesCikar() {
    return `${super.sesCikar()} Hav hav!`; // super ile üst metoda erişim
  }
}

7. instanceof Operatörü

Bir nesnenin belirli bir sınıfın örneği olup olmadığını kontrol eder. 

const k = new Kopek("Karabaş", " Kangal");
console.log(k instanceof Kopek);  // true
console.log(k instanceof Hayvan); // true
