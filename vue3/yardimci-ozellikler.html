<script setup>
import { reactive } from 'vue'

const user = reactive({
  name: 'Sergen'
})

  user.name = 'Mehmet'
}

  'Elma',
  'Armut'

toRef()

toRef(), bir objenin tek bir özelliğini ref haline getirir. Ama yeni bir ref oluşturmaz, orijinal property'ye bağlı (linked) bir ref oluşturur. Bir objenin tek bir değerine reactive bozmadan destructuring yaparak ref ile tanımlanmış gibi kullanabiliriz. 



<script setup>
import { ref, toRef } from 'vue'

const user = ref({
  name: 'Sergen',
  age: '25'
})

function changeName() {
  user.value.name = 'Mehmet'
}

const isim = toRef(user.value, 'name')
const age = toRef(user.value, 'age')


</script>

<template>

  <h1>{{ isim  }} - {{ age  }}< /h1>


  <button @click="changeName">
    Değiştir
  </button>

</template>

toRefs

toRef in yaptğının aynısını çoklu versiyon şeklinde yapar. her birini tek tek yazma durumunda kalmadan yapar. 


<script setup>
import { ref, toRefs } from 'vue'

const user = ref({
  name: 'Sergen',
  age: 25
})

function changeName() {
  user.value.name = 'Aslan'
  user.value.age = 27
}

const { name, age } = toRefs(user.value)

</script>

<template>

  <h1>{{ name  }} - {{ age }}</h1>


  <button @click="changeName">
    Değiştir
  </button>

</template>

readonly

bir ref veya reactive ile tanımlanmış bir değişkeni sadece okunabilir ve değeri değiştirilemez şekle dönüştürmek için kullanılan bir özellik. Değeştirmeye çalışıldığında hata verecektir. 



<script setup>
import { ref, readonly } from 'vue'

const count = ref(0)

const readonlyCount = readonly(count)
 
</script>

<template>

  <h1>{{ readonlyCount  }} </h1>


  <button @click="readonlyCount++">
    Değiştir
  </button>

</template>

shallowRef()

  • çok büyük obje verilerinin (veryBigProductList) içerisindeki verileri proxy ile kaplamamak ve performansı artırmak için kullanılır.
  • Normalde ref() içine bir obje koyarsan Vue objenin tüm içindeki property'leri reactive yapar.
  • shallowRef() ise yalnızca en dıştaki (top-level) değeri reactive yapar.
  • ref() .value değişirse Vue bunu algılar. ✅
  • shallowRef() .value içindeki property'ler değişirse Vue bunu izlemez. ❌ 


const user = ref({
    name: 'Sergen'
})

user.value.name = 'Ali' -- güncellenir ✅

const user = shallowRef({
    name: 'Sergen'
})

user.value.name = 'Ali' --güncellenmez ❌


const users = shallowRef(veryBigProductList)

users.value = {data} -- güncellenir ✅

triggerRef()

shallowRef() ile tanımlanan değişkenler içersindeki veride güncelleme olduğunda vue render etmez tepki vermez bunun için el ile vue ya render yaptırabilmemizi sağlar. 


  
const user = shallowRef({
    name: 'Sergen'
})

user.value.name = 'Ali'
triggerRef(user) --vue böyle yapınca render eder
