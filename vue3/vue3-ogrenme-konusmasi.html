2. script setup öğren

Vue 3’ün gerçek olayı burada.

Vue 2:

export default {
  data() {},
  methods: {},
  computed: {}
}

Vue 3:

<script setup>
import { ref } from 'vue'

const count = ref(0)

function increment() {
  count.value++
}
</script>

Asistan

Destructuring reactivity neden bozar?

Bu Vue 3’te çok kritik bir nokta. Vue 2 biliyorsan ilk başta tuzak gibi gelir.

Problem ne?

Vue 3 reactivity, objenin property erişimini proxy ile takip eder.

const state = reactive({
  name: 'Sergen',
  age: 30
})

Burada:

state.name

Vue bunu izler. Çünkü erişim hâlâ state objesi üzerinden.

Destructuring yapınca ne oluyor?

const { name } = state

Artık name, state ile bağlantılı reactive property değil. Sadece düz bir değer olur.

console.log(name)

Bu artık şöyle davranır:

const name = 'Sergen'

Örnek

Hatalı:

const state = reactive({
  count: 0
})

const { count } = state

state.count++

console.log(count)

Beklenti:

1

Gerçek:

0

Çünkü destructuring bağlantıyı kopardı.

Neden?

Çünkü bu:

state.count

proxy getter'dan geçer.

Ama bu:

count

artık proxy ile alakalı değildir.

Vue'nun takip ettiği şey:

state -> count

Ama sen şunu yaptın:

const count = state.count

Vue artık bunu izleyemez.

Çözüm 1: toRefs()

Reactive objeyi parçalayacaksan:

import { reactive, toRefs } from 'vue'

const state = reactive({
  count: 0,
  name: 'Sergen'
})

const { count, name } = toRefs(state)

Artık:

count.value

reactive kalır.

state.count++
console.log(count.value)

Çıktı:

1

Çözüm 2: toRef()

Tek property lazımsa:

const count = toRef(state, 'count')

ref'te durum farklı mı?

Evet.

const count = ref(0)

Bunda zaten primitive ref taşıyorsun. Destructuring konusu objelerde problem.

props'ta da aynı tuzak var

const props = defineProps({
  title: String
})

Şunu yaparsan:

const { title } = props

reactivity bozulur.

Doğrusu:

const { title } = toRefs(props)

veya modern Vue’de:

const title = toRef(props, 'title')
Kısa kural: reactive() objesini direkt destructure etme.
Reactive kalmasını istiyorsan toRefs() / toRef() kullan.
