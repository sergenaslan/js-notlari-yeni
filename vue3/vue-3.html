reactivity Bir verinin değiştiğini Vue’nun otomatik fark edip ekrandaki ilgili yerleri güncellemesi.

vue2 syntaxt vue3 sytaxt

ref

ref, Vue 3 Composition API'de reaktif (reactive) veri tanımlamak için kullanılan bir fonksiyondur. Bir değişkenin değeri değiştiğinde Vue'un bunu fark edip ekranı güncellemesini sağlar.

ref bir nesne döndürür ve gerçek değere .value ile ulaşılır


<script setup>
  import { ref } from 'vue'
  const count = ref(0)
</script>

<template>
  <div>
    <p>Sayı: {{ count }}</p>
    <button @click="count.value++">Arttır</button>
   <button @click="count.value--">Azalt</button>
  </div>
</template>
