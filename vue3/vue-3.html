vue3 sytaxt

import { ref } from 'vue'

const count = ref(0)
function increment() {
  count.value++
}

ref

ref, Vue 3 Composition API'de reaktif (reactive) veri tanımlamak için kullanılan bir fonksiyondur. Bir değişkenin değeri değiştiğinde Vue'un bunu fark edip ekranı güncellemesini sağlar.

ref bir nesne döndürür ve gerçek değere .value ile ulaşılır


<script setup>
  import { ref } from 'vue'
  const count = ref(0)
</script>

<template>
  <div>
    <p>Sayı: {{ count }}</p>
    <button @click="count.value++">Arttır</button>
   <button @click="count.value--">Azalt</button>
  </div>
</template>

reactive

Bir nesneyi (object, array, Map, Set) reaktif hale getirmek için kullanılır.

reactive değişkenin değeri değiştiğinde Vue bunu takip eder ve ilgili component yeniden render edilir.


 <script setup>
import { reactive } from 'vue'

const user = reactive({
  name: 'Sergen',
  age: 30
})

function changeName() {
  user.name = 'Aytaç'
}

</script>

<template>
  <h1> {{user.name }} </h1>
  <button @click="changeName">İsmi Değişitir</button>
</template>

computed

vue3 de computed kullanımı aşşağıdaki gibidir

Bir değeri başka değerlerden hesapla ve bu hesaplamayı Vue takip etsin. ve içerisindeki değişken değiştiğinde yeniden hesaplansın 


<script setup>
 
import { ref, computed } from 'vue'

const firstName = ref('Sergen')
const lastName = ref('Aslan')

const fullName = computed(() => {
  return `${firstName.value} ${lastName.value}`
})
function changeName() {
   firstName.value  = 'Aytaç'
}
</script>

<template>
  <h1> {{fullName }} </h1>
  <button @click="changeName">İsmi Değişitir</button>
</template>

watch

vue3 de wathc kullanımı aşşağıdaki örnekteki gibidir


<script setup>
 
import { ref, computed, watch } from 'vue'

const firstName = ref('Sergen')
const lastName = ref('Aslan')

const fullName = computed(() => {
  return `${firstName.value} ${lastName.value}`
})
function changeName() {
   firstName.value  = 'Aytaç'
}

watch(fullName, (newValue, oldValue) => {
  console.log('Eski:', oldValue)
  console.log('Yeni:', newValue)
})



</script>

<template>
  <h1> {{fullName }} </h1>
  <button @click="changeName">İsmi Değişitir</button>
</template>

watchEffect

watchEffect in watch dan farkı şudur, hangi değişkenleri takip etmesi gerektiğiniz biz söylemeyiz. kendisi watchEffect içerisinde kullanılan değişkenlere bakarak takip etmesi gereken değişkenleri belirler 


<script setup>

import { ref, watchEffect } from 'vue'

const firstName = ref('Sergen')

watchEffect(() => {
  console.log('1111111111111111111111111111')
  console.log(firstName.value)
})

</script>

<template>
  <h1> {{fullName }} </h1>
  <button @click="changeName">İsmi Değişitir</button>
</template>

onMounted, onUnmounted

onMounted, onUnmounted özelliklerinin vue3 de kullanımı


<script setup>

import { onMounted, onUnmounted } from 'vue'

onMounted(() => {
  console.log('Component yüklendi')
})

onUnmounted(() => {
  console.log('Component kaldırıldı')
})



const handleResize = () => {
  console.log(window.innerWidth)
}

onMounted(() => {
  window.addEventListener('resize', handleResize)
})

onUnmounted(() => {
  window.removeEventListener('resize', handleResize)
})

</script>
