defineProps

defineProps() aslında Vue 2'deki props tanımının Vue 3 <script setup> içindeki karşılığıdır.

  • defineProps() → Parent'ten gelen verileri tanımlar.
  • Sadece <script setup> içinde kullanılır.
  • Reactive'dir.
  • Child component props'u okuyabilir ama değiştiremez.
  • Vue 2'deki props:{} yapısının Vue 3 Composition API karşılığıdır. 

<script setup>
const props = defineProps({
  name: {
    type: String,
    required: true
  }
  age: Number,
  isAdmin: {
    type: Boolen,
    default: false
  }
})

</script setup>

<UserCard
  name="Sergen"
  :age="30"
  :is-admin="true"
/>

Template tarafında kebab-case, JavaScript tarafında camelCase kullanılır. is-admin _> isAdmin

defineEmits

defineEmits aslında Vue 2'deki $emit kullanımının Composition API tarafındaki tanımlama şeklidir.


  --temel kullanım
<script setup>
const emit = defineEmits(['save'])

function save() {
  emit('save')
}
</script>

<script setup>

const emit = defineEmits(['save'])

const saveUser = () => {
  emit('save', {
    id: 1,
    name: 'Sergen'
  })
}

</script>
