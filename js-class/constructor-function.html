Constructor Function

Constructor Function Nedir?

Constructor function, JavaScript'te benzer özelliklere sahip birden çok nesne (object) oluşturmak için kullanılan bir şablondur. Normal fonksiyonlardan farkı, new anahtar kelimesiyle çağrılmaları ve this anahtar kelimesini kullanmalarıdır. 


const user1 = {
  name: "Ali",
  age: 25
};

const user2 = {
  name: "Ayşe",
  age: 30
};
yukarıdaki gibi tek tek yazmak yerine aşşağıdaki gibi bir constructor function tanımlaması ile 
istenilen sayıda nesne üretilebilnir.

function User(name, age) {
  this.name = name;
  this.age = age;
}

const user1 = new User("Ali", 25);
const user2 = new User("Ayşe", 30);

console.log(user1); // User { name: 'Ali', age: 25 }
console.log(user2); // User { name: 'Ayşe', age: 30 }

Mantığı nedir?

  • ortak yapıya sahip nesneler üretmek
  • kod tekrarını azaltmak
  • her nesneye özellik ve metot verebilmek

Prototype ile Metot Paylaşımı

Her nesnede aynı fonksiyonu tekrar tekrar oluşturmamak için metotları prototype'a eklemek daha verimlidir.


function Kisi(isim, yas) {
    this.isim = isim;
    this.yas = yas;
}

// Metodu prototype'a ekleme (tüm nesneler paylaşır)
Kisi.prototype.selamVer = function() {
    console.log(`Merhaba, ben ${this.isim}`);
};

const ali = new Kisi("Ali", 25);
const ayse = new Kisi("Ayşe", 30);

ali.selamVer(); // Merhaba, ben Ali
ayse.selamVer(); // Merhaba, ben Ayşe

// İkisi de aynı fonksiyonu kullanır (hafıza tasarrufu)
console.log(ali.selamVer === ayse.selamVer); // true

Yazım ve Kullanım Kuralları

Constructor Functionların ilk harfi büyük olarak yazılır. Kullanıldığı yerde new anahtar kelimesi ile kullanılır ve normal fonksiyon kullanımından ayırılan noktası burasıdır. 


function User(name, age) {
  this.name = name;
  this.age = age;
}

const user1 = new User("Ali", 25);
