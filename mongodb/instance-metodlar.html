INSTANCE METHODLAR

1. INSTANCE METHODLAR (document metodları)

Bunlar new User() ile oluşturduğun objelerde olur.


const user = new User({ name: "Sergen" });

bunlar user instance üzerinde çalışır.

user.save()         → Veriyi MongoDB’ye kaydeder
user.remove()       → Bu document’i siler
user.deleteOne()    → Bu document’i siler
user.updateOne()    → Sadece bu document’i günceller
user.toObject()     → Plain JS objesine çevirir
user.toJSON()       → JSON’a çevirir
user.validate()     → Schema kurallarına göre doğrular
user.populate()     → Referans alanlarını doldurur (ref)
user.isModified()   → Alan değişmiş mi kontrol eder
