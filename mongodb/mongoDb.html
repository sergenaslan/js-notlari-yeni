mongoose.model()

model fonskiyonu shemayada bakarak 1. parametredeki isimle bir class oluşturur

User adında bir class (constructor function) oluşturur

  • MongoDB’deki users collection’ına bağlıdır
  • İçinde schema kuralları vardır
  • Static + instance methodları vardır 

const User = mongoose.model("User", schema);

Yani zihninde şöyle canlandır:
class User {
  constructor(data) {
    // schema’ya göre data’yı işler
  }

  save() {}
  static find() {}
}
